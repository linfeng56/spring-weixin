
# 工作计划管理
  一个工作计划系统的基本功能。可以对每周进行工作项进行计划，对每周的进行工作总结。
可以对周计划进行导出。

## 推荐运行配置
1. JDK 8
2. Tomcat 8.5
3. MySQL 5.7
4. CentOS 7
5. Maven 3.6
